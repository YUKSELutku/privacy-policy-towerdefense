This Privacy Policy explains how the Meme Monster Defense mobile application, referred to as the “Application,” handles information.

Meme Monster Defense is a mobile tower defense game. The Application does not require users to create an account, does not ask for a name, email address or phone number, and does not operate a developer backend that stores user profiles.

1. Information Processed by the Application

Local game data

The Application stores gameplay and preference data locally on the user's device. This may include:

  • Game progress, completed levels, stars and scores.
  • Coins, gems, chests, cards and unlock progress.
  • Unlocked towers, skills and monster files.
  • Tower and skill upgrade levels.
  • Survival mode records.
  • Tutorial completion status.
  • Language selection.
  • Music and sound effect volume settings.

This information is stored on the device using local application storage. Yuksel Software does not receive or maintain this gameplay data on its own servers.

Device permissions

The current version of the Application does not request access to precise location, camera, microphone, contacts, calendar, photos or files.

The Application does not request background location permission and does not track the user's physical location.

Audio and visual assets

Game images, sound effects and music are bundled with the Application or loaded from the Application package. Playing game audio does not require microphone access and does not record the user.

2. Advertising and Rewarded Ads

The Application includes game screens that may support rewarded advertising features, such as continuing a survival run or doubling a level reward. In the current implementation, these features may be unavailable unless an advertising provider is enabled in a future release.

If rewarded advertisements or another advertising provider are enabled in a future version, third-party advertising SDKs may process information such as:

  • IP address, which may be used to estimate approximate location.
  • Advertising identifiers, such as the Android Advertising ID.
  • Application interactions, ad impressions, taps and reward events.
  • Device, diagnostic and performance information.

If advertising SDKs are added or enabled, this Privacy Policy will be updated to identify the advertising provider and describe the relevant data processing more specifically.

4. Data Collection and Sharing Summary

For Google Play Data Safety purposes, the current version of the Application is designed to store gameplay data locally on the user's device and does not require users to submit personal information to Yuksel Software.

  • Personal information: The Application does not ask users to provide a name, email address or phone number.
  • Location: The Application does not request precise or approximate device location.
  • Game progress: Stored locally on the device for gameplay functionality.
  • Device identifiers: Not collected by Yuksel Software in the current version. Advertising identifiers may be processed by an advertising provider if ads are enabled in a future version.
  • Diagnostics: Not collected by Yuksel Software in the current version. Platform providers may process technical information according to their own policies.

Yuksel Software does not sell users' personal information.

5. User Choices

Users can control or remove locally stored information through Android system settings:

  • Clear the Application's storage to remove local progress and settings.
  • Uninstall the Application to remove local application data.
  • Change language, music and sound settings inside the Application.
  • Reset or delete the Android Advertising ID from Android privacy settings, if advertising is enabled.

Clearing local storage or uninstalling the Application may permanently remove saved game progress from the device.

6. Data Retention and Deletion

Local gameplay progress and preferences remain on the device until the user clears the Application's storage or uninstalls the Application.

Because Yuksel Software does not operate user accounts for this Application, Yuksel Software generally cannot restore deleted local progress or delete local data remotely.

Information processed by Google Play or other third-party providers is retained according to those providers' own policies.

7. Data Security

The Application uses standard Android and Flutter platform mechanisms for local storage. Reasonable efforts are made to keep the Application secure. However, no method of electronic storage or transmission can be guaranteed to be completely secure.

8. Children's Privacy

The Application does not knowingly request personal information directly from children. Users are not required to create an account or provide contact information to play the game.

Parents or guardians who believe that a child has provided personal information to Yuksel Software may contact us using the email address below.

9. International Processing

The Application itself stores gameplay information locally on the user's device. Google Play and any third-party providers may process information in countries other than the user's country of residence, according to their own privacy policies and applicable safeguards.
